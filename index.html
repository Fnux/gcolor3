Gcolor3 - Color picker

Choose colors from the picker or the screen

Download latest Older releases View on GitHub

Ever wanted to know the value of that color on your screen? Gcolor3 enables you to pick the color from any pixel on your screen. It also offers a palette, so that you can easily mix and match a couple of colors together.

When you have found the perfect combination of colors, naturally you want to save them. Gcolor3 allows you to conveniently save and retrieve colors.

Gcolor3 is a color selection dialog written in GTK+ 3. It is much alike Gcolor2, but uses the newer GTK+ version and other modernisations to better integrate into your modern desktop. It has the same feature set as Gcolor2, except that recent versions of Gcolor3 use an .ini style file to save colors, in a location compliant to the XDG directory specification.

Versions

The most recent version of Gcolor3 requires GTK+ 3.16. If your distribution uses an older version, please see the older releases.

Translations

You can help translating Gcolor3 to your own language! Currently, Gcolor3 is available in Dutch, American and British English, French, Galician, German, Greek, Serbian, Swedish and Ukranian. New translations are always welcome! For instructions on how to translate Gcolor3, see the project README.
