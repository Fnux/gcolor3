Gcolor3

A simple color chooser written in GTK+ 3

Download latest Older releases View on GitHub

Gcolor3 is a color selection dialog written in GTK+ 3. It is much alike Gcolor2, but uses the newer GTK+ version to better integrate into your modern desktop. It has the same feature set as Gcolor2, except that recent versions of Gcolor3 use an .ini style file to save colors (older versions use the same file as Gcolor2).

Versions

The most recent version of Gcolor3 requires GTK+ 3.12. If your distribution uses an older version, please see the older releases. Functionally, all releases are the same; the only exception is that since version 2.0 Gcolor3 uses an .ini style format to save colors.

Translations

You can help translating Gcolor3 to your own language! Currently, Gcolor3 is available in American and British English, Dutch, French, Greek, German and Swedish. New translations are always welcome! For instructions on how to translate Gcolor3, see the project README.
